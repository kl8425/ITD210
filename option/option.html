CSS Option

My website topic is Different Cheeses. Below is a list of the six CSS categories that I used.

  1. Multi-column layout: Used on the “Cheese Types” section of the home page to display cheese descriptions in two columns.
  2. Horizontal scrolling: Used in the “Cheese Gallery” section so images of cheeses scroll horizontally on mobile.
  3. CSS attribute selectors: Used on external links with [target="_blank"] to style them orange with underlines.
  4. Structural pseudo-classes: Used :nth-child(odd) on the cheese cards to give alternating background colors.
  5. @font-face (Google Font): Imported a custom font for headings to make the cheese names stand out.
  6. Alpha transparency: Used RGBA on the hero banner overlay for a transparent background behind the cheese title.
